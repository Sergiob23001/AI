# Repository Structure

```
ai-product-roadmap/
├── README.md              ← Main roadmap (renders on GitHub)
├── roadmap-visual.html    ← Visual portfolio page (open in browser)
└── STRUCTURE.md           ← This file
```

## How to use

- **README.md** — Reads directly on GitHub. Share the repo link on LinkedIn.
- **roadmap-visual.html** — Open locally in any browser for a visual version. Can also be hosted on GitHub Pages.

## GitHub Pages (optional)

To host the visual version online:
1. Go to your repo → Settings → Pages
2. Source: Deploy from branch → `main` → `/root`
3. Your visual roadmap will be live at `https://yourusername.github.io/ai-product-roadmap/roadmap-visual.html`
