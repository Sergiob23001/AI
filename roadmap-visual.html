01
Weeks 1–4 · Discovery & Foundation
Define Scope & Validate Data Readiness
Goal: Align stakeholders, map data sources, establish AI governance framework

Key Deliverables

  • Stakeholder alignment — define "good AI output" criteria
  • Data source mapping: XLS ingestion, diagnostic outputs, historical benchmarks
  • Human oversight framework for consultant review/override
  • RACI across engineering, AI/ML, UX, client teams
  • Sprint 0: environment setup, model API integration, CI/CD

AI-Specific Risks

Hallucination Risk
GPT summarizations may introduce inaccuracies in consultant-facing narratives — define accuracy threshold before launch
Data Quality Risk
Inconsistent XLS schemas from client uploads will degrade benchmark accuracy
Scope Creep
AI feature requests arrive ambiguously — weekly backlog refinement is mandatory
02
Months 1–3 · Pilot & Iteration
Deploy, Capture Feedback, Iterate Fast
Goal: Ship GPT summarization MVP to pilot clients, measure accuracy, close feedback loop

Sprint Cadence

  • S1–S2: Data pipeline + XLS ingestion hardening
  • S3–S4: GPT summarization MVP + UX integration
  • S5–S6: Benchmark intelligence v1 (financial services vertical)
  • S7–S8: Feedback capture, regression testing, pilot launch
  • Quarterly: Onsite workshop in Prague with core stakeholders

Key Deliverables

  • Consultant feedback loop — thumbs up/down + free-text annotation UI
  • Model performance dashboard: accuracy, hallucination rate, latency
  • Prompt version registry for regression tracking
  • Model drift monitoring — GPT updates changing behavior without code changes
03
Months 3–9 · Expansion
Scale Validated Features & Add ROI Automation
Goal: Multi-vertical expansion, ROI automation, compliance hardening for enterprise accounts

Key Deliverables

  • ROI calculation automation — AI-generated baseline reports within platform
  • Multi-vertical benchmark expansion: telecoms, retail, healthcare
  • A/B testing framework for prompt variants
  • Audit trail & compliance hardening for PwC enterprise accounts
  • Self-service reporting — consultants generate reports without PM intervention

Months 9–18 · Optimization & Scale
Enterprise Rollout & Next Horizon Scoping
Goal: Full large-account rollout, measure business outcomes, define next AI investments

Key Deliverables

  • Large account onboarding — complex, high-stakes environments
  • Business outcome tracking: turnaround time ↓, consultant hours ↓, ROI accuracy ↑
  • Model fine-tuning cycles based on 6+ months annotated feedback
  • Knowledge base: prompt libraries, lessons learned, delivery playbooks

Next-Horizon AI Scoping

  • Predictive diagnostics — proactive issue identification before client reports
  • Voice analytics integration — call center audio → automated diagnostic input
  • Real-time benchmark updates — live KPI comparison vs. industry data streams
  • Multi-language support — expand beyond English for EU client base
